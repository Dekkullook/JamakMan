# Jamak Man

영상과 자막의 이름을 통일 시켜줍니다. (기준은 영상 파일의 이름)  
딱히 라이센스는 없습니다.

## [사용법]

1. Run jar file.
> java -jar Jamakman ver0.01.jar

2. Enthe Target Path
> d:\directory

## [주의 사항]

1. 영상과 자막의 개수가 다르면 작동하지 않습니다.
2. 영상과 자막은 같은 폴더(Target Path) 에 있어야 합니다.
3. 기본적으로 () [] 와 같은 파일이름의 seed데이터를 제거합니다.
